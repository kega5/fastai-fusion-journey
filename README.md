# fastai-fusion-journey
Learning to apply Machine Learning to nuclear fusion.

A learning log and code repo tracking my progress through fast.ai and applied machine learning, building toward my master's thesis at the University of Saskatchewan Plasma Physics Laboratory: Machine Learning-Enhanced Langmuir Probe Array Diagnostics for Edge Turbulence Studies on the STOR-M Tokamak.

The goal of this repo is to:


1. Build genuine ML literacy by adapting core concepts toward plasma physics and diagnostics problems wherever possible.
2. Keep a transparent, dated record of progress, including the rough early stuff.


Background

BSc Physics, Ontario Tech University. Starting MSc at USask (Fall 2026), working on edge turbulence diagnostics using Langmuir probe arrays on STOR-M. Long-term interest in machine learning applications for fusion diagnostics and control, with an eye toward roles at places like CNL, General Fusion, or Commonwealth Fusion Systems.
